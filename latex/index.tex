The BMP280 is a digital sensor that measures temperature and pressure. This library provides a convenient interface to interact with the sensor using the I2C communication protocol. It includes functions to set the power mode, configure oversampling settings, read raw sensor data, and calculate compensated temperature and pressure values.

Took inspiration from\+: Bosch\textquotesingle{}s BMP280 API\+: \href{https://github.com/BoschSensortec/BMP2-Sensor-API/}{\texttt{ https\+://github.\+com/\+Bosch\+Sensortec/\+BMP2-\/\+Sensor-\/\+API/}} wovo\textquotesingle{}s example code\+: \href{https://github.com/wovo/hwlib-examples/tree/master/arduino-due/due-\%230210-dmp280}{\texttt{ https\+://github.\+com/wovo/hwlib-\/examples/tree/master/arduino-\/due/due-\/\%230210-\/dmp280}} basvandenbergh\+: https\+: \href{https://github.com/BasvandenBergh/IPASS_jaar1_BAS}{\texttt{ https\+://github.\+com/\+Basvanden\+Bergh/\+IPASS\+\_\+jaar1\+\_\+\+BAS}}
