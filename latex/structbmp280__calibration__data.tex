\doxysubsection{Detailed Description}
Struct that holds calibration data for the BMP280 sensor. 

This struct holds the calibration data for the BMP280 sensor. It includes the calibration coefficients for temperature and pressure measurements, as well as the t\+\_\+fine value used for temperature calculation.
